three_point_one - directory containing all of the Java code for the LRParser
                  for grammar 3.1
g3.1.txt - The file containing grammar 3.1 that was given to the program to use
           to generate a parse table.  It starts with a list of terminals
           and is followed by the grammar rules.  Since the original grammar
           was ambiguous, this grammar is an equivalent one that isn't
           ambiguous.
3_1_parse_table.txt - the parse table for grammar 3.1 generated by our program.
sample_parse.txt - the parse for the program:  x  :=  (print(y), 2 + 3)
sample_program.txt - a file containing the program:  x  :=  (print(y), 2 + 3)
