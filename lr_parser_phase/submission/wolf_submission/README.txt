wolf - directory containing all of the Java code for the LRParser
wolf.txt - the grammar for our language that was given to the program to use
           to generate a parse table.  It starts with a list of terminals
           and is followed by the grammar rules.
wolf_parse_table.txt - the parse table for WOL(F) generated by our program.
parses - the directory containing sample parses of all of the sample programs
sample_programs - the programs that the LR Parser was tested on.
